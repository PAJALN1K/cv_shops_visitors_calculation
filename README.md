# Проект: Анализ посещаемости магазинов с помощью нейронных сетей в компьютерном зрении

Данный проект позволяет анализировать посещаемость магазинов, используя записи с веб-камер и нейронную сеть YOLO (You Only Look Once) для обработки видеопотока.

## Использование

### 1. Создание виртуального окружения

В зависимости от вашей операционной системы выполните следующие действия:

- **Windows**:

    ```bash
    python -m venv env
    env\Scripts\activate
    ```

- **Linux/Mac**:

    ```bash
    python3 -m venv env
    source env/bin/activate
    ```

### 2. Установка зависимостей

Скачайте необходимые библиотеки, запустив команду:

```bash
pip install -r requirements.txt
```

### 3. Подготовка входных данных

Загрузите свои видеофайлы с записями с веб-камер и создайте конфигурационный файл shops_config.json, содержащий данные о магазинах. Пример структуры файла:

```json

[
    {
        "name": "Shop x",
        "pts": [[35, 390], [160, 370], [170, 590], [55, 615]]
    },
    ...
]
```

_Где name — это название магазина, а pts — координаты точек, определяющих зону, в которой будет вестись подсчет посещаемости._

### 4. Запуск анализа

Начните работу с Jupyter Notebook, открыв файл src/main.ipynb, где вы найдете код для выполнения анализа.
